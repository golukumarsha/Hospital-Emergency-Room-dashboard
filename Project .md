# 🚑 Hospital Emergency Room Dashboard

## 📌 Project Overview
This project is an interactive **Hospital Emergency Room Dashboard** created using **Excel**.  
The dashboard helps analyze hospital emergency room data and provides meaningful insights for better decision-making.

I learned this dashboard design and healthcare data analysis concepts with guidance from **Satish Dhawale**.

## 📂 Dataset Used

[📥 Download Dataset](https://github.com/golukumarsha/Hospital-Emergency-Room-dashboard/blob/main/Golu%20kumar.xlsx)

---

## 📊 Dashboard Features

### ✅ Patient Admission Status
- Shows the number of:
  - Admitted Patients
  - Not Admitted Patients

### ✅ Patient Age Distribution
- Groups patients according to different age categories.

### ✅ Timeliness Analysis
- Measures the percentage of patients attended within **30 minutes**.

### ✅ Gender Analysis
- Displays patient distribution by gender.

### ✅ Department Referrals
- Shows which hospital departments receive the highest referrals.

### ✅ Additional Insights
- Average Wait Time
- Patient Satisfaction Score
- Monthly Emergency Room Report
